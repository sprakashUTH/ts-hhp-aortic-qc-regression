# Draft data and code availability statement

The Python code used to perform data quality control, longitudinal modeling, progression classification, and participant ranking is available in a public repository at [repository URL], archived at [DOI]. The repository includes the final analysis scripts, software environment files, input data dictionary, QC workflow documentation, and instructions for regenerating the analytic outputs.

Because the underlying analytic dataset contains participant-level clinical and imaging data from a rare-disease cohort, individual-level data are not publicly posted. A de-identified analytic dataset or limited dataset may be made available to qualified investigators after approval by the appropriate data-governance body and completion of a data use agreement, consistent with participant consent and institutional review board requirements.
