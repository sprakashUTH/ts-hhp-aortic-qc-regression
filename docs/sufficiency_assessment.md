# Sufficiency Assessment

The uploaded sources are sufficient to reconstruct and document:

- baseline aortic diameter by segment
- best-model slope analyses by segment
- floored slope analyses
- linear diameter-growth regressions
- left-censored Tobit analyses of floored growth
- mixed-effects time-by-baseline-diameter interaction models
- progression >=1 mm/year logistic models using HHP covariates
- early growth regression datasets
- AHI/delta risk visualizations

## Covariates successfully annotated

This archive annotates:
- age at first HHP study
- BAV
- coarctation
- hypertension
- BMI
- 45,X and non-45,X karyotype indicators
- prior surgery coding from HHP

The uploaded files do not contain the manuscript follow-up endpoint components:
- death
- aortic dissection
- follow-up elective aortic surgery
- composite endpoint date/status

Therefore, the clinical endpoint regression analyses cannot be fully reconstructed from these files alone. An additional deidentified endpoint file keyed by `patient_id` or enriched-subset record ID would be needed.
